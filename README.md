# *safe_belt*
## 한이음, 안전벨트 착용감지 앱프로젝트.

### Why did i choose this.
- 다인승의 차량에 안전벨트 착용여부의 지속적 확인으로 승객의 안전의식 강화 목적.

### Development Environments.
- Android
- Arduino

### Configuration.
1. 승객들이 차량에 탑승.
2. 착석
2.1 안전벨트를 착용 후 운전, 정상.
2.2 안전벨트를 착용하지 않고 운전, 해당 좌성에 Beep음 출력.

### What did i learn.
- Android의 기본적인 개발 방법.
- Arduino의 기본 개요, 저항, 하드웨어의 응용 방법.
- Android와 Arduino의 연동.
